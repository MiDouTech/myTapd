# Task006：企业微信深度集成

> **业务模块**：工单系统  
> **依赖**：Task004、Task005  
> **预估工时**：6天  
> **对应产品文档**：4.6.3 企微应用消息推送、4.6.4 企微群机器人工单

---

## 一、任务目标

实现企微应用消息推送（工单事件通知到个人企微）、企微群机器人回调服务（@机器人创建工单）、群 Webhook 状态推送，完成企微深度集成闭环。

## 二、交付物清单

| 序号 | 交付物 | 路径/说明 |
|------|--------|-----------|
| 1 | 企微回调入口 | WecomCallbackController（URL 验证 + 消息接收） |
| 2 | 回调异步处理 | 签名验证、AES 解密、MsgId 去重、5 秒内返回、消息入 MQ |
| 3 | 消息解析器 | WecomBotMessageParser（缺陷模板、通用模板） |
| 4 | 群机器人创建工单 | @工单助手 #分类 标题 格式解析，调用工单创建 API |
| 5 | 群机器人指令 | 帮助、分类、查询工单、我的工单、催办 |
| 6 | 企微应用消息推送 | 工单创建、分派、状态变更、SLA 预警/超时、催办、评论@提及 |
| 7 | 群 Webhook 推送 | 工单状态变更推送到关联群 |
| 8 | 群与分类绑定 | wecom_group_binding 配置管理 |
| 9 | RabbitMQ 消费者 | WecomMessageConsumer 异步处理消息并回复 |
| 10 | 消息模板 | 文本卡片、Markdown 格式 |

## 三、接口清单（需填接口编号）

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 企微回调 | GET/POST | /api/wecom/callback | URL 验证、消息接收 |
| 群绑定列表 | GET | /api/wecom/group-binding/list | 群绑定配置列表 |
| 群绑定创建 | POST | /api/wecom/group-binding/create | 新增群绑定 |
| 群绑定更新 | PUT | /api/wecom/group-binding/update/{id} | 修改群绑定 |

## 四、验收标准

- [ ] 企微回调 5 秒内响应，业务异步处理  
- [ ] 群内 @机器人可按模板创建工单，并收到回复  
- [ ] 工单事件能推送到个人企微与关联群  
- [ ] 缺陷工单群内模板解析正确  

## 五、产出说明

企微集成完成后，用户可通过企微群快速建单，并实时接收工单通知。
