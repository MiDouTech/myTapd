# Task020：系统设置增强与集成配置

> **业务模块**：工单系统（前端）  
> **依赖**：Task016、Task017、Task006、Task010  
> **预估工时**：2天  
> **对应产品文档**：4.6.4 企业微信深度集成、4.11 开放能力、5.2 管理模块

---

## 一、任务目标

在 Task016 的系统设置基础版上，完成“Webhook 配置管理 + 企微群绑定配置”两项可运维能力，形成系统管理员可操作、可维护、可追踪的集成配置中心。

## 二、功能范围

### 2.1 Webhook 配置管理

- 配置列表分页展示
- 新增、编辑、删除配置
- 配置字段：URL、密钥、事件类型、超时、重试次数、启用状态、描述
- 展示最近成功/失败时间与失败原因（如后端返回）

### 2.2 企微群绑定管理

- 展示群绑定列表（群ID、名称、分类、模板、状态）
- 新增群绑定、修改群绑定
- 支持按分类筛选绑定配置

### 2.3 系统设置页结构收口

- 将系统设置页按“基础配置/通知配置/集成配置”分组
- Webhook 与企微群绑定统一纳入“集成配置”Tab
- 保留配置说明与风险提示（例如 URL 校验、密钥保密）

## 四、详细实施内容

### 4.1 Webhook 管理子模块

- 列表页字段排序与筛选（按状态、事件类型）
- 新增/编辑使用弹窗表单，字段级校验：
  - URL 必须为 http/https
  - timeout/retry 为正整数
  - 事件类型至少选择一项
- 删除需二次确认，防止误删

### 4.2 企微群绑定子模块

- 列表支持展示关键绑定关系，避免只显示 ID
- 新增/编辑表单支持分类下拉、模板下拉
- 对群ID唯一性冲突做前端提示（以后端校验为准）

### 4.3 系统设置体验优化

- 提供“最近变更时间”“变更人”展示区（如接口可返回）
- 表单保存后自动刷新列表并高亮最近变更项（增强体验）
- 统一错误提示与提交 loading

## 八、风险与应对

- **风险1：配置错误影响线上推送**  
  处理：增加前端校验与二次确认；建议后端提供“测试连通性”接口（后续可扩展）。

- **风险2：配置字段多导致可用性下降**  
  处理：分组展示表单字段，提供示例与说明文案。

- **风险3：群绑定依赖外部系统变更**  
  处理：对“群不存在/权限不足”错误做明确提示并保留表单状态。
